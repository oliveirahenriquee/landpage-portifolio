Dashboard digital


A Dashboard servirá como uma forma de gráfico para mostrar a produtividade do usuário, contendo informações como tarefas diárias, rotina de estudos, alimentação e exercícios físicos.


Tabela interativa de rotina semanal


Tabela com todos os dias da semana e horários de preferência do usuário, cada célula é editável para que as informações sejam inseridas, editadas ou excluídas, contendo botões para salvar alterações, inserir informações e excluir dados.


Formulário de cadastro.


Sistema simples de cadastro com dados pessoais e caixa de mensagens para descrição ou sugestões.
